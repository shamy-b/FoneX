# FoneX

FoneX is a modern e-commerce website for purchasing smartphones online. It offers a clean, user-friendly interface to browse and shop for the latest mobile phones from top brands like Apple, Samsung, Google, Nothing, and Oppo.

## Features

- **Home Page** - Featured products showcase with horizontal and vertical product cards
- **Product Catalog** - Browse all available smartphones with detailed information
- **Search Functionality** - Search for specific phone models
- **User Authentication** - Login and user profile management
- **Shopping Cart** - Cart management and cart history
- **About Us** - Learn about the FoneX team and mission
- **Contact Us** - Multiple contact options including phone, email, social media, and location

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with responsive design
- **JavaScript** - Client-side functionality
- **Font Awesome 6.5.2** - Icons

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open `index.html` in your web browser

3. Navigate through the pages using the navigation menu
